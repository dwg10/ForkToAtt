• The total value of all balls of one colour can be calculated by multiplying the quantity of balls of that colour with the relevant value.
  • In this example, this would mean that the red balls in the first basket have a total value of quantity x value = 40 x 2 = 80.
  • Adding these total values for all three colours within a basket will give you the total value of a basket.
  • In the example, this would mean that the first basket has a total value of (4 x 5) + (40 x 2) + (10 x 1) = 110.
  • The second basket has a total value of (10 x 5) + (24 x 2) + (2 x 1) = 100.
  • This means that the first basket is more valuable than the second basket.
